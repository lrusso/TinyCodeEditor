# Tiny Code Editor

## Web

https://tinycodeeditor.lrusso.com.ar

## Controls
| Action  | Windows/Linux | Mac |
| ------------- | ------------- | ------------- |
| New document  | CTRL+N | CMD+N |
| Open document  | CTRL+O  | CMD+O |
| Save document  | CTRL+S  | CMD+S |
| Print document  | CTRL+P  | CMD+P |
| Find  | CTRL+F  | CMD+F |
| Find and replace  | CTRL+R  | CMD+R|
