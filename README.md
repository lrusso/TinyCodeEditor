# Tiny Code Editor

## Online Web:

https://tinycodeeditor.lrusso.com.ar

### Controls (Windows/Linux):

CTRL+N = New document

CTRL+O = Open document

CTRL+S = Save document

CTRL+P = Print document

CTRL+F = Find

CTRL+R = Find and replace

### Controls (Mac):

CMD+N = New document

CMD+O = Open document

CMD+S = Save document

CMD+P = Print document

CMD+F = Find

CMD+R = Find and replace
